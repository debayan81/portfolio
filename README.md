# Debayan Das - Portfolio

A clean, responsive single-page developer portfolio built with plain HTML, CSS, and vanilla JavaScript. No build step, no dependencies.

**Live site:** https://debayan81.github.io/portfolio/

## Features
- Responsive layout (desktop → mobile) with a hamburger menu on phones
- Light / dark mode toggle (remembers your choice, respects system preference)
- Sections: About, Education, Experience, Projects, Skills, Achievements, Contact
- Resume download button and social links

## Development
It's a single file - just open `index.html` in a browser. To edit, change `index.html` and refresh.

## Deployment
Hosted on GitHub Pages from the `main` branch.
